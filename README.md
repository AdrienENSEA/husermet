# Projet Logiciel Transversal

## USE
```console
$ git clone https://github.com/Solanna13/husermet
$ cd husermet
$ mkdir build
$ cd build
$ cmake ..
$ make
```
S'il y a des problèmes avec OpenGL, il faut désactiver l'accélération 3D dans les paramètres d’affichage de la VM. <br>
<br>
